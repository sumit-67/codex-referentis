Trigonometric Functions

  • \( \sin^2{\theta} + \cos^2{\theta} = 1 \)
  • \( \sin{ \left( - \theta \right) } = -\sin{\theta} \)
  • \( \cos{ \left( - \theta \right) } = \cos{\theta} \)
  • \( \cos{ \left( \alpha \pm \beta \right)} = \cos{\alpha} \cos{\beta} \mp \sin{\alpha} \sin{\beta} \)
  • \( \sin{ \left( \alpha \pm \beta \right)} = \sin{\alpha} \cos{\beta} \pm \cos{\alpha} \sin{\beta} \)
  • \( \tan{ \left( \alpha \pm \beta \right) } = \dfrac{ \tan{\alpha} \pm \tan{\beta} }{1 \mp \tan{\alpha} \tan{\beta}} \)
  • \( \cot{ \left( \alpha \pm \beta \right) } = \dfrac{1 \mp \cot{\alpha} \cot{\beta}}{ \cot{\alpha} \pm \cot{\beta} } \)
  • \( 2 \cos{\alpha} \cos{\beta} = \cos{ \left( \alpha + \beta \right) + \cos{ \left( \alpha - \beta \right)} } \)
  • \( - 2 \sin{\alpha} \sin{\beta} = \cos{ \left( \alpha + \beta \right) - \cos{ \left( \alpha - \beta \right)} } \)
  • \( 2 \sin{\alpha} \cos{\beta} = \sin{ \left( \alpha + \beta \right) + \sin{ \left( \alpha - \beta \right)} } \)
  • \( \sin{2\theta} = 2 \sin{\theta} \cos{\theta} \)
  • \( \cos{2\theta} = \cos^2{\theta} - \sin^2{\theta} \)
  • \( \sin{3\theta} = 3\sin{\theta} - 4\sin^3{\theta} \)
  • \( \cos{3\theta} =4\cos^3{\theta} - 3\cos{\theta} \)

Approximations

\(0 < \alpha \ll 1 \)

  • \( \left( 1 \pm \alpha \right) ^ n \approx 1 \pm n \alpha \)
  • \( e^\alpha \approx 1 + \alpha \)
  • \( \ln{\left( 1 + \alpha \right)} \approx \alpha \)
  • \( \sin{ \alpha } \approx \alpha \)
  • \( \cos{ \alpha } \approx 1 - \alpha^2 / 2 \)
  • \( \tan{ \alpha } \approx \alpha \)

Expansions

  • \( \left( 1 - x \right)^n = 1 -\dfrac{nx}{1!} + \dfrac{n \left( n-1 \right)}{2!} - \cdots\) (Binomial Theorem \( x^2 < 1 \))
  • \( e^x = 1 + x + \dfrac{x^2}{2!} + \dfrac{x^3}{3!} + \cdots = \displaystyle \sum_{n=0}^{\infty} \dfrac{x^n}{n!} \)
  • \( \ln{\left( 1+x \right)} = x - \dfrac{x^2}{2} + \dfrac{x^3}{3} - \cdots = \displaystyle \sum_{n=1}^\infty {\left( -1 \right)}^{n+1} \dfrac{x^n}{n} \)
  • \( \sin{\theta} = \theta - \dfrac{\theta^3}{3!} + \dfrac{\theta^5}{5!} - \cdots = \displaystyle \sum_{n=1}^\infty {\left( -1 \right)}^{n+1} \dfrac{\theta^{2n - 1}}{\left( 2n - 1 \right)!} \)
  • \( \cos{\theta} = 1 - \dfrac{\theta^2}{2!} + \dfrac{\theta^4}{4!} - \cdots = \displaystyle \sum_{n=0}^\infty {\left( -1 \right)}^{n} \dfrac{\theta^{2n}}{\left( 2n \right)!} \)
  • \( \tan{\theta} = \theta - \dfrac{\theta^3}{3!} + \dfrac{2\theta^5}{15!} + \dfrac{17\theta^7}{315!} + \cdots \)
